思考

知识点

  • 变量声明赋值，变量类型
  • 数据类型：基本类型值和引用类型值
  • 赋值变量：对于引用类型复制的是值
  • 编译与执行，内存运行机制
  • 赋值都是从右向左进行赋值
  • 变量的定义：松散类型能保存任何数据类型的值， 保存值得占位符,在内存中有地址。

a.x = a = {n:2}

  1. 在解析中，从左至右，此时的a指向{n:1}，解析器之后会在堆内存的{n：1}中暂时生成一个属性x，但是指向的是null, 此时x 在堆中有个地址。
  2. 变量a 改变指向，指向新的{n:2}。
  3. a.x = a, 这行的意义是，{n:1}.x = a ，解析器是按照刚才那个堆中地址索引进行赋值。并不是按照a.x， 因为这时的a已经改变了指向，改变后的 a.x 并不存在
  4. b 是指向 {n：1}, 进行了上述的执行，b ={n:1, x:{n:2}}
